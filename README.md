# A House of Prayer

Interactive world prayer map built as a single-page HTML app.

## GitHub Pages

This repository is ready to publish from the repository root:

1. Push these files to a GitHub repository.
2. Open the repository on GitHub.
3. Go to Settings -> Pages.
4. Under "Build and deployment", choose "Deploy from a branch".
5. Select the `main` branch and `/ (root)`.
6. Save.

GitHub will publish the site at:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/`

## Files

- `index.html` - the complete interactive map.
- `.nojekyll` - tells GitHub Pages to serve files directly without Jekyll processing.
